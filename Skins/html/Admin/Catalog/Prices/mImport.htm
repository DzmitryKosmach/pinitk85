Импорт прайса:



Файл с прайсом (xls/xlsx):
Необходимо загрузить Excel-файл, идентичный по структуре файлу, загружаемому через Экспорт прайсов (нельзя добавлять/удалять/перемещать/переименовывать колонки таблицы).

- В прайсе можно изменить любые данные, кроме категории серий, а также ID серий/товаров.
- Идентификация серий и товаров производится по их ID.
- При считывании цен и наценок для каждого товара либо вычисляется наценка (по вход./выход. ценам), либо выход.цена (по вход.цене и наценке) - это зависит от соответствующей опции каждой серии.
- Для обозначения валюты используйте форматирование ячеек: любой формат с символом «$» принимается как У.Е., остальное считается рублями.
