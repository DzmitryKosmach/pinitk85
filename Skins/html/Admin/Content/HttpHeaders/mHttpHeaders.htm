Настройка автоматических редиректов (с кодом ответа 301) и ошибок (код 404)



Каждая команда - редирект или ошибка - с новой строки.

Для настройки редиректа нужно указать URL, с которого происходит редирект, и URL, на который происходит редирект, разделив их знаком «=» ("равно")
Пример:
http:///catalog/stoly/ = http:///catalog/stoly-i-stulya/

Для того, чтобы по определённому адресу возвращалась ошибка 404, нужно указать URL, знак «=» ("равно") и число «404»
Пример:
http:///catalog/stoly/ = 404

ВАЖНО: Каждая команда срабатывает не только для URL'а слева от знака «=», но также и для адресов, начинающихся с указанного URL.
Например, если указано:
http:///catalog/stoly/ = 404
То для всех подразделов этой страницы также будет выдаваться «ошибка 404», т.е. отдельно здесь перечислять их не нужно
